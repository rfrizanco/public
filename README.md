Privacy Policy – ToxicCleaner for Instagram

ToxicCleaner for Instagram is a browser extension designed to help users identify Instagram accounts they follow that do not follow them back.

Data Collection
This extension does not collect, store, transmit, or share any personal or sensitive user data.

Local Processing
All analysis of followers and following relationships happens locally within the user's browser. No data is sent to external servers.

User Preferences
The extension stores only minimal user preferences locally using the browser storage API. This may include:
- The last username analyzed
- A list of accounts the user chose to ignore

These preferences remain on the user's device and are never transmitted externally.

Third-Party Services
This extension interacts only with Instagram pages when the user explicitly initiates the analysis.

Affiliation
This extension is not affiliated with, endorsed by, or officially connected to Instagram.
